# ZBS CRM Bot 🚀

Telegram-бот для управления контентом, клиентами, задачами и финансами ZBS Media.

## Возможности

### 📅 Контент-календарь
- Планирование контента по дням, типам (подкаст, видео, reels, кружки, посты) и платформам
- Назначение ответственных
- Просмотр: сегодня / неделя / мой контент
- Изменение статуса: план → в работе → проверка → опубликовано

### 👥 CRM — Клиенты и сделки
- База клиентов с контактами
- Sales pipeline: лид → переговоры → КП → договор → в работе → завершено
- Суммы сделок и визуализация воронки

### 📋 Задачи
- Создание задач с приоритетами (🟢🟡🟠🔴)
- Назначение на участников команды
- Дедлайны с автоматическими напоминаниями
- Отслеживание просроченных задач

### 💰 Финансы
- Учёт прихода/расхода по категориям
- Отчёт по проектам
- Месячная сводка

### 📊 Автоматизация
- **09:00** — Утренний отчёт дня (админам)
- **10:00** — Напоминания о дедлайнах (исполнителям)
- **11:00** — Алерты о просроченных задачах (админам)
- **20:00** — Вечерняя проверка контента

## Деплой на Railway

### 1. Создай бота
- Напиши @BotFather → /newbot → получи токен

### 2. Создай проект на Railway
```
railway init
```

### 3. Добавь PostgreSQL
- Dashboard → New → Database → PostgreSQL
- Railway сам подставит DATABASE_URL

### 4. Задай переменные окружения
```
BOT_TOKEN=your_bot_token
ADMIN_IDS=271065518
TZ=Asia/Tashkent
```

### 5. Деплой
```
railway up
```

Или через GitHub: подключи репозиторий в Railway dashboard.

## Команды бота

| Команда | Описание |
|---------|----------|
| /start | Запуск и регистрация |
| /menu | Главное меню |
| /today | Контент на сегодня |
| /mytasks | Мои задачи |
| /addtask | Быстро создать задачу |
| /addcontent | Добавить контент в план |
| /report | Отчёт дня |
| /help | Справка |

## Роли

- **👑 Админ** — полный доступ + управление командой
- **📋 Менеджер** — управление + отчёты
- **👤 Участник** — свои задачи и контент

## Стек
- Python 3.12
- aiogram 3.x (Telegram Bot API)
- SQLAlchemy + asyncpg (PostgreSQL)
- APScheduler (cron-задачи)
- Docker (Railway deployment)
